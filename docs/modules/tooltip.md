## API

参数 | 说明 | 类型 | 默认值 | 
-- | -- | -- | -- | 
maxWidth | 提示框内容最大宽度，单位px | number | 120 | 
content | 展示的文本 | string | '暂无内容' | 
title | 提示的文本 | string | '暂无提示' | 
fontSize | 提示文本字体大小，单位px | number | 14 | 
color | 提示文本字体颜色 | string | '#FFF' | 
backgroundColor | 提示框背景色 | string | 'rgba(0,0,0,.85)' |
