## API

| 参数 | 说明 | 类型 | 默认值 |
|---|---|---|---|
| type | 配色类型 | normal | success | error | warning | info | normal |
| round | 是否圆角 | boolean |  |
| to | 跳转链接 | string | # |
| bold | 文字是否加粗 | boolean | false |
